## **SECTION 2: Problems / Smells**

### **[HIGH] Critical Issues**
- **Duplicate CTA Components:** Two different CTA implementations (`/components/CTA.js` vs `/components/HomeSections.js`)
- **20+ Missing CSS Classes:** References to undefined CSS classes causing styling failures (`page-header`, `contact-form`, etc.)
- **Client-Side Form API:** Direct Formspree call exposes endpoint, no server validation

### **[MEDIUM] Moderate Issues**
- **Unused ScrollReveal Component:** 47 lines of dead code
- **Duplicate Portfolio Data:** Projects defined in both `/lib/visuals.js` and `/app/work/[slug]/page.js`
- **Unoptimized Gallery Images:** Using `unoptimized={true}` flag defeats Next.js image optimization
- **Hardcoded Case Studies:** 250+ lines of inline data in dynamic route file

### **[LOW] Minor Issues**
- **Mixed Styling Patterns:** Inline styles vs Tailwind (inconsistent)
- **Missing Contact Metadata:** Contact page lacks SEO metadata
- **Naming Inconsistencies:** Different names for similar concepts (hero/page-header, etc.)
